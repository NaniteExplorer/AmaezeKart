# 🛒 Amaezekart

**Production-Grade E-Commerce Platform (MERN Stack)**

Amaezekart is a full-stack e-commerce application designed with a focus on **scalability, clean architecture, and real-world production workflows**.
It goes beyond a typical CRUD project by integrating authentication, payments, state management, and modular backend design.

---

## 🚀 Live Demo

🔗 *buy-genius-amaze-kart.vercel.app*

---

## 🧠 Engineering Highlights

* **Modular Backend Architecture**
  Clean separation of concerns (routes, controllers, services, models)

* **Authentication & Security**
  JWT-based authentication, password hashing, protected routes

* **Payment Integration**
  Integrated **Stripe** for secure and scalable transactions

* **State Management**
  Efficient global state handling using **Redux**

* **Scalable API Design**
  RESTful APIs designed for extensibility and maintainability

* **Cloud Media Handling**
  Image uploads and optimization using **Cloudinary**
